Description

Project Leo involves a Raspberry Pi running Raspbian and some Python and Bash scripts. The end goal is to turn it into somewhat of a personal assistant that can send weather information, bank balances, perform tasks, and respond to text messages.

Features

Weather:

sends a text message with a summary of weather information, including current/high/low temperatures and a short summarizing statement of the day's conditions.

Balance (In Progress):

uses a mint.com account to retrieve bank balance information and sends a text message to the user with a summary.

Text Message Input (In Progress):

responds to text messages and performs the task corresponding to the message.

More to come!
